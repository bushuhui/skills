# PyWord

基于现有 Python 脚本体系的 Word `.docx` 自动化开发与复用技能。

## 快速开始

AI 使用本 Skill 时，按以下顺序查阅文档：

1. **先读** `docs/01-quick-index.md` - 通过关键字或场景快速定位功能
2. **再看** `docs/03-capabilities.md` - 确认功能完整度等级和边界
3. **后查** `docs/02-modules.md` - 参考可抽取函数和组合示例
4. **最后** 查阅 `scripts/` 中的实际脚本代码
