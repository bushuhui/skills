# 功能脚本概览

## 文档创建与基础编辑 (01.xx)

| 脚本 | 功能描述 |
|-----|---------|
| `01.01_创建新文档_将进酒.py` | 创建空白文档，写入《将进酒》全文，设置默认字体格式 |
| `01.02_结构化生成三维重建报告.py` | 创建结构化报告，按节点递归写入标题和正文 |
| `01.03_段落插入与红色修订.py` | 追加段落、在指定位置插入段落、段落内追加文本，并设置红色修订格式 |
| `01.04_基础标题插入与编号.py` | 插入一级/二级/三级标题，创建编号规则，绑定样式与编号体系 |
| `01.05_多方式删除段落.py` | 按 id、文本、路径、索引删除段落，输出删除报告 |

## 文档内容提取与解析 (02.xx)

| 脚本 | 功能描述 |
|-----|---------|
| `02.01_提取全文文本.py` | 提取正文纯文本，识别段落样式和标题路径，输出 JSON |
| `02.02_解析文档结构.py` | 识别标题层级，构建文档结构树，输出结构树和扁平清单 |
| `02.03_多规则提取摘要正文.py` | 按关键字、正则、标题路径提取摘要等多种规则 |
| `02.04_标题诊断与规范化.py` | 诊断标题级别、编号连续性、格式符合性，输出诊断报告 |
| `02.05_读取表格与绑定表题.py` | 读取表格内容，绑定表题，输出 JSON/CSV |
| `02.06_提取文档图片与清单.py` | 提取 media 图片，统计格式和引用位置，输出清单 |
| `02.07_提取审阅内容.py` | 提取批注、修订等审阅内容，输出 JSON/CSV |
