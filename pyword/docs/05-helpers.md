# 辅助模块详细说明

## _01_docx_bookmarks.py（书签管理）

1. 生成符合 Word 书签命名规则的书签名称，并自动清洗非法字符与长度。
2. 读取文档中已有书签编号，并计算下一个可用书签 `id`。
3. 向整段段落插入书签起止节点，建立段落级定位锚点。
4. 向指定 `run` 前后插入书签起止节点，建立片段级交叉引用锚点。

---

## _02_docx_captions.py（图题/表题）

1. 创建图题、表题等题注段落，并写入"标签 + 自动序号 + 标题文本"结构。
2. 插入 `SEQ` 序号域，实现题注编号的自动更新。
3. 插入题注书签，供后续交叉引用或目录生成使用。

---

## _03_docx_fields.py（域代码）

1. 创建 Word 复杂域代码，并按 `begin / instrText / separate / end` 结构写入域节点。
2. 插入中文或英文目录域，支持按指定标题级别生成目录。
3. 插入图目录、表目录等分类目录域。
4. 插入 `SEQ`、`REF`、`PAGEREF` 等序号与交叉引用域。
5. 设置文档打开时自动更新域，便于目录、页码和交叉引用刷新。

---

## _04_docx_math.py（公式）

1. 创建 OMML 公式文本节点，写入 Word 原生公式内容。
2. 创建上标结构节点，生成如 `c²` 一类公式片段。
3. 创建分式结构节点，生成如 `a/b` 一类公式片段。
4. 组合公式元素并插入段落，生成完整的 Word 原生公式块。

---

## _05_docx_numbering.py（编号）

1. 读取现有编号定义并生成新的 `abstractNumId`。
2. 创建单级或多级编号定义，写入 `numbering.xml` 的层级规则。
3. 设置编号格式、编号文本、缩进、悬挂缩进和项目符号字体。
4. 向段落写入编号属性，实现列表、标题等段落的自动编号。

---

## _06_docx_styles.py（样式）

1. 读取、创建并获取段落样式，统一管理文档样式对象。
2. 设置中英文字体、字号、对齐、行距、段前段后、字符缩进和大纲级别。
3. 设置标题续段、孤行控制等版式约束。
4. 应用论文标题样式和扩展样式规则，批量准备常用样式体系。
5. 设置正文默认样式，统一文档基础字体和段落格式。

---

## _07_docx_tables.py（表格）

1. 设置整张表格的上边框、下边框、内横线、内竖线等边框规则。
2. 隐藏整张表格边框，生成无边框布局表格。
3. 应用三线表边框样式，并为首行额外设置分隔线。
4. 设置单元格文字的字体、字号、加粗与段落对齐方式。

---

## _08_docx_textops.py（文本操作）

1. 遍历正文、表格、页眉和页脚中的全部段落。
2. 克隆原始 `run` 样式并重建文本片段，尽量保持原有格式结构。
3. 解析段落中各 `run` 的文本区间，定位待替换文本的字符范围。
4. 在不直接覆盖整段文本的前提下替换段落中的目标内容。
5. 批量替换正文、表格、页眉、页脚中的指定文本。

---

## _09_format_presets.py（格式预设）

1. 定义常用字号预设，如小二、三号、四号、小四、五号。
2. 定义宋体、仿宋、黑体、Times New Roman 等字体族映射。
3. 定义对齐方式、行距模式、段前段后单位和缩进单位预设。
4. 解析字号、字体和对齐方式预设，供格式脚本统一调用。

---

## _10_style_presets.py（样式预设）

1. 定义论文正文、一级标题、二级标题、三级标题等样式规则。
2. 定义引导样式、引用样式、说明样式等自定义样式规则。
3. 定义多级标题编号规则与自动格式化正则规则。
4. 解析字体、字号、对齐方式并生成样式标识。

---

## _11_docx_enhance_presets.py（扩展预设）

1. 定义目录标题、图题、表题、公式编号、参考文献、列表正文等扩展样式规则。
2. 提供图片示例路径、表格示例数据和参考文献示例数据。
3. 提供中文标点替换规则，供标点规范化脚本调用。
4. 提供审查修正规则，供自动修正脚本调用。

---

## _12_docx_format_helpers.py（格式辅助）

1. 创建格式脚本共用的测试目录定位能力。
2. 设置默认正文样式、字体字号和中英文字体映射。
3. 设置直接格式、字符边框和段落边框。
4. 设置行距、段前段后、首行缩进、悬挂缩进和字符缩进。
5. 解析并重建文本片段 `run` 结构，对指定文本片段执行局部格式化。

---

## _13_docx_title_helpers.py（标题辅助）

1. 识别一级、二级、三级和三级以下标题的文本模式。
2. 识别图题、表题、目录项等需要排除的非正文章节段落。
3. 创建四级及以下标题样式，并为多级标题脚本提供统一规则。
4. 规范段落文本空白，供标题识别和模板规范化脚本复用。

---

## _14_docx_review_demo.py（审阅示例）

1. 创建文档安全与审阅示例源文档。
2. 插入示例标题、正文和表格内容，供密码保护与评论脚本复用。
3. 统一输出测试目录和示例源文档路径，避免 19.x 脚本重复创建演示文档。

---

## _15_docx_crypto.py（加密）

1. 实现 OOXML Agile Encryption 所需的密钥派生、AES-CBC 加解密和 HMAC 完整性校验。
2. 构建 `EncryptionInfo` XML 与 `EncryptedPackage` 二进制流，生成可用于打开密码保护的加密包。
3. 写入最小可用的 CFB 容器，封装 `EncryptionInfo` 与 `EncryptedPackage` 流。
4. 读取已生成的 CFB 容器并解密回 `.docx` 包内容，用于脚本自校验。

---

## _16_docx_revisions.py（修订）

1. 提供修订相关的 OOXML 包读写、`trackRevisions` 设置和 `document.xml` 更新能力。
2. 生成插入、删除、移动等修订包装节点，并支持按段落写入修订内容。
3. 接受或拒绝 `w:ins`、`w:del`、`w:moveFrom`、`w:moveTo` 等常见文本修订节点。
4. 提供文本分词与差异分段能力，供文档 compare/merge 脚本生成带修订的差异段落。
