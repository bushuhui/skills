# 功能总体概览

本 Skill 提供基于 Python + `python-docx` 的 Word `.docx` 自动化处理能力，包含文档创建、内容编辑、格式设置、样式系统、目录、表格、图片、公式、书签、交叉引用、文本替换、审阅保护、文档提取与统计分析等完整功能。

---

## 核心功能领域

### 1. 文档创建与副本管理
- 新建空白文档
- 文档复制与模板克隆
- 模板注入与占位符替换

### 2. 内容生成与基础编辑
- 段落插入、删除
- 标题插入与多级编号
- 分页符与分节符

### 3. 格式与样式控制
- 字体字号设置
- 段落格式（对齐、行距、缩进）
- 直接格式（加粗、斜体、下划线、颜色）
- 边框设置（字符边框、段落边框）
- 样式系统构建与应用

### 4. 标题、列表、目录与编号
- 多级标题结构
- 自动编号（1 / 1.1 / 1.1.1）
- 项目符号列表与数字编号列表
- 中文目录、英文目录、图目录、表目录

### 5. 表格、图片、公式与题注
- 表格插入与三线表设置
- 图片插入与图题
- Word 原生 OMML 公式与公式编号

### 6. 书签、域代码与交叉引用
- 书签插入（段落级、片段级）
- 目录域、SEQ 序号域、REF 交叉引用域、PAGEREF 页码域

### 7. 提取与解析文档内容
- 正文提取与结构解析
- 多规则文本提取
- 表格、 图片提取
- 审阅内容提取

### 8. 识别与规范文档结构
- 标题识别与诊断
- 标题规范化
- 模板规则应用
- 现有段落自动格式化

### 9. 统计分析
- 文档属性读取
- 字数统计（总体与指定章节）
- 参考文献年份统计
- 近五年中英文文献统计

### 10. 安全与审阅
- 限制编辑密码保护
- 打开文档密码加密
- 审阅评论写入与提取
- 修订接受/拒绝

### 11. 文档比较与合并
- 旧稿新稿差异比较
- 带修订合并文档生成
- 接受/拒绝合并结果输出
